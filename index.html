The plugin is designed to quickly create an input mask. All that is needed - assign mask in the data attribute data-mask of elements or in options mask at initialize the plugin.

+38 (068) 732-84-22
+38 ({d3}) {d3}-{d2}-{d2}

648-G6hY2-gHH
{d3}-{*5}-{w3}

Initialize:

Options
var options = {
  maks: "+38 ({d3}) {d3}-{d2}-{d2}",
  placeholder: "Phone number"
}
$(selector).inputMask( options );

or

Element attributes
data-placeholder="Phone number"
data-mask="+38 ({d3}) {d3}-{d2}-{d2}"
$(selector).inputMask();

Options:

  1. mask - a input mask has syntex {[d|w|*][0-9]}.
    The first part indicates the type of values ​​supported.
    • d - only numbers [0-9]
    • w - only word character [a-zA-Zа-яА-ЯіІїЇєЄёЁ]
    • * - number and word character [a-zA-Zа-яА-ЯіІїЇєЄёЁ0-9]
    The second - number of characters.
  2. placeholder - this option will then be absolutely positioned over the text input, so that it displays over the top when the input has no value, and is not focused.
